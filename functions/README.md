**Firebase Function**

This boilerplate provide sample function & testing

**Testing**

```
npm test
```

**Build**

Dev
```
npm run serve:dev
```

Production
```
npm run build
```

**Deploy**

```
npm run deploy
```

**Firebase Config**

Set config

```
firebase functions:config:set someservice.key="THE API KEY" someservice.id="THE CLIENT ID"
```

Get config

```
firebase functions:config:get
```
